# Local Build Protocol

## Production Build
```
git clone https://github.com/andrewderango/IceAnalytics
cd IceAnalytics
npm ci
npm run build
serve -s build
```

Then visit http://localhost:3000. This should be copied to your clipboard when served.

## Development Server
```
git clone https://github.com/andrewderango/IceAnalytics
cd IceAnalytics
npm ci
npm start
```

Then visit http://localhost:3000.
