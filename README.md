IceAnalytics is a predictive simulation engine for the NHL that utilizes an ensemble machine learning model to project player statistics and estimate probability distributions. 

Projection engine that simulates thousands of NHL seasons to project team and player performances, and generate probability distributions for various individual and team statistics.
